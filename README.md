# Disbyte Engine

## Overview
A high-level 3D Game Engine written in C++17 and using modern graphic, sound and script libraries.

## Platforms
  * Windows 7
  * Windows 8/8.1
  * Windows 10

## Compilers
  * Visual Studio 2015
  * Clang 3.5
  * GCC 5.0

## Dependencies
  * AcousticsLib
  * libSDL2pp
  * SOL2
  * SPDLog
  * PugiXML
  * nvAPI
  * AMD AGS SDK
